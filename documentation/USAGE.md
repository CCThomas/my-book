# Usage

## Setup
- Download Docker
- Download Project
  - git clone git@github.com:CCThomas/my-book.git


## Run
- Spin up Database, Backend, and Frontend
  - `docker-compose up`

## Clean Up
- Nuke Docker
  - Create an executable
    - `chmod +x docker-nuke.sh`
  - Run docker nuke
    - `./docker-nuke.sh`
